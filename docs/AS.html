AS | Android Studio is the official Integrated Development Environment (IDE) for Android app development, built on JetBrains IntelliJ IDEA. In FTC, Android Studio is used to write, build, and deploy Java or Kotlin robot control code to the Robot Controller phone or Control Hub. It provides tools like Gradle-based build automation, an emulator, debugging, and version control integration. While many teams start with Blocks or OnBot Java, Android Studio offers full flexibility for advanced teams — including custom libraries, multi-module projects, and direct hardware abstraction layer access through the FTC SDK.
