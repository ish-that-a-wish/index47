GITHUB
GitHub is a web-based platform built on top of Git that hosts code repositories and enables version control, collaboration, and project management. In FTC, teams use GitHub to store and share their robot code, track changes across the season, and collaborate between programmers. Key features include branches for testing new code without breaking the main project, pull requests for reviewing teammates' changes, and GitHub Pages for hosting team websites. The official FTC SDK is hosted on GitHub, and teams typically fork it as the starting point for their season's codebase.
